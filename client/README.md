# The Ask Pete Initiative: LDT Master's Portfolio

* **Author:** Joshua Atkinson
* **Stack:** React, Tailwind, Framer Motion
* **Purpose:** Comprehensive evidence submission for the Purdue LDT Competency Badge system.

## Overview

This repository contains the source code for the "Ask Pete" Master's Portfolio, a competency-based portfolio designed for the Learning Design and Technology program at Purdue University.

The project demonstrates "Systems Isomorphism" by aligning the technical architecture (Rust, React, Local-First) with the pedagogical philosophy (Cognitive Logistics, Data Sovereignty).

## Key Features

* **Competency Portfolio:** A structured display of artifacts mapped to LDT badges.
* **Ask Pete Initiative:** A white paper and technical breakdown of the "Living Laboratory."
* **Digital Archive:** A direct file access system for raw artifacts.
* **Thematic Design:** A "Black and Gold" theme with "Daydream" aesthetics.

## Deployment

This project is designed to be deployed on Vercel or similar static hosting platforms.

```bash
npm install
npm run build
```
